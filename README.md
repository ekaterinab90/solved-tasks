#### % % %

````javascript
function percent(price){
  if (price>= 300){
    return price - price * 0.3 // aici 3 inseamna 30 %
  }else if(price>= 200){
    return price - price * 0.2; //aici 2 inseamna 20 % 
 }else if(price>= 100){
    return price - price * 0.1; //aici 1 inseamna 10 % 
 }
}


````
